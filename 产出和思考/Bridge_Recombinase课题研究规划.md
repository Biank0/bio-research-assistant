# Bridge Recombinase 课题研究规划

> 训练家小边 × 沙奈朵
> 创建时间：2026-06-02
> 状态：初始规划

---

## 总体目标

以 IS110 家族桥接重组酶为研究对象，通过 **复现验证 → 新系统发现 → AI 蛋白工程** 三条主线，开发活性更高、特异性更好的可编程 DNA 重组工具。

## 总体逻辑

```
主线一：复现验证（打地基）
  → 建立原核评测平台
  → 获得 ISCro4 / IS621 基线数据
          ↓
主线二：生信筛选新亚型              主线三：AI 改造蛋白
  → 数据库挖掘                     → 蛋白语言模型预测
  → 扩大候选库                     → 结构引导理性设计
  → 发现新特性                     → 定向进化 + AI 迭代
          ↓                               ↓
          └──────────── 汇合 ─────────────┘
                         ↓
          最优系统 = 最佳天然亚型 × AI 优化蛋白 × 增强型 bRNA
                         ↓
                推进至哺乳动物细胞验证
```

## 主线一：复现与验证

### 1.1 原核 reporter 系统构建
- [ ] 在大肠杆菌中搭建倒位 / 切除 / 插入三种 reporter
- [ ] 设计标准化量化方法（PCR + 流式 + 测序）
- [ ] 建立阳性对照（已知活性系统）和阴性对照

### 1.2 ISCro4 原核活性验证
- [ ] 测试天然 bRNA 活性
- [ ] 测试 split bRNA（拆分 TBL + DBL）
- [ ] 测试 enhanced bRNA（stem 优化、G:C 增强等）
- [ ] 量化三种操作（插入 / 倒位 / 切除）的效率
- [ ] 时间梯度实验：重组反应动力学

### 1.3 IS621 平行对照
- [ ] 与 ISCro4 使用相同 reporter、相同靶位点
- [ ] 定量比较两者在原核中的效率差异
- [ ] 确认人类细胞中的优势是否在原核中也成立

### 1.4 关键残基解析
- [ ] 针对 ISCro4 vs IS621 的 39 个氨基酸差异位点
- [ ] 构建嵌合体：IS621 框架 + ISCro4 关键残基（尤其是 7 个核酸结合位点附近残基）
- [ ] 反向嵌合体：ISCro4 框架 + IS621 残基，验证是否降低活性
- [ ] 确定最小增强突变集

### 1.5 蛋白-RNA 兼容性实验
- [ ] ISCro4 蛋白 + IS621 bRNA → 能否工作？
- [ ] IS621 蛋白 + ISCro4 bRNA → 能否工作？
- [ ] 揭示蛋白与 bRNA 之间的互换规则

### 1.6 核心序列偏好探索
- [ ] CT 核心突变为其他二核苷酸（CA、CG、CC 等），测活性
- [ ] 确认不同亚型是否有不同核心偏好
- [ ] 评估核心序列限制对可靶向范围的影响

## 主线二：生物信息学筛选新亚型

### 2.1 数据库挖掘
- [ ] 以 ISCro4 / IS621 蛋白序列做 BLAST / HMMer 搜索（NCBI nr/nt）
- [ ] 以 bRNA 保守结构做 Infernal / covariance model 搜索
- [ ] 以 IS110 转座酶结构域做 Pfam / InterPro 搜索
- [ ] 扩展至宏基因组数据库（MGnify、IMG/M）
- [ ] ISfinder 数据库系统检索

### 2.2 候选筛选标准
- [ ] 蛋白序列一致性梯度分布（50%-90%）
- [ ] bRNA TBL / DBL 环结构可识别性（RNAfold 预测）
- [ ] 天然靶位点 / 供体位点序列特征提取
- [ ] 宿主物种的系统发育多样性

### 2.3 重点筛选维度
- [ ] 核心序列多样性 → 有没有不用 CT 核心的？（扩大靶向范围）
- [ ] 识别长度 → 有没有 >14 nt 的？（天然更高特异性）
- [ ] 蛋白大小 → 有没有 <300 aa 的？（有利于递送）
- [ ] 反应方向性 → 有没有天然偏好插入的？

### 2.4 系统发育分析
- [ ] 构建 IS110 家族重组酶进化树（全局 + 关键结构域）
- [ ] 分析活性 vs 进化距离的关系
- [ ] 识别"功能聚类"——哪些进化分支更可能在异源宿主中工作

### 2.5 高通量实验验证
- [ ] 选 50-100 个候选，合成重组酶 + bRNA
- [ ] 在大肠杆菌 reporter 中批量筛活性
- [ ] 设立分级标准：初筛（有/无活性） → 复筛（定量效率） → 精筛（特异性）

## 主线三：AI 辅助蛋白工程

### 3.1 数据基础准备
- [ ] 整理 Perry 论文深度突变扫描数据（~6,200 个单突变体活性值）
- [ ] 整理 ISCro4 结构信息（AlphaFold 预测 + Pelea 2.8 Å cryo-EM）
- [ ] 收集主线二新亚型的序列，构建多序列比对（MSA）
- [ ] 标注已知功能残基：催化位点、RNA 结合、DNA 结合、四聚体界面

### 3.2 蛋白语言模型（PLM）应用
- [ ] 用 ESM-2 / ProtTrans 对 IS110 重组酶做 embedding
- [ ] 结合深度突变扫描数据训练活性预测回归模型
- [ ] 预测组合突变（双突变、三突变）的活性
- [ ] 与随机组合做对比，验证模型预测能力

### 3.3 结构引导理性设计
- [ ] 蛋白-bRNA 界面 → 优化 RNA 结合亲和力的突变
- [ ] 蛋白-DNA 界面 → 优化催化效率的突变
- [ ] 四聚体蛋白-蛋白界面 → 优化突触复合物稳定性的突变
- [ ] 用 AlphaFold3 / RosettaFold 预测突变体结构，计算结构稳定性变化

### 3.4 减少 DBL-DBL 交叉重组的设计
- [ ] 分析 DBL-DBL 重组的结构基础：哪些残基允许两个 DBL 替代 TBL+DBL？
- [ ] 设计破坏 DBL-DBL 但保留 TBL+DBL 正常组装的突变
- [ ] 提高特异性而不显著降低活性

### 3.5 AI + 定向进化迭代
- [ ] 第一轮：AI 预测 top 100 变体
- [ ] 第二轮：原核 reporter 实验验证
- [ ] 第三轮：实验数据反哺模型，迭代优化
- [ ] 重复 2-3 轮直至收敛

### 3.6 bRNA 的 AI 优化
- [ ] 用 RNA 结构预测工具（RhoFold / RNAstructure）评估 bRNA 变体的热稳定性
- [ ] 设计更稳定的 enhanced bRNA scaffold
- [ ] 探索 TBL / DBL 环长度与识别特异性的关系

## 需要确认的问题

1. 当前实验条件：能做原核？能做哺乳动物细胞？有高通量测序资源吗？
2. 主线优先级：三条线同时推进还是先聚焦某一条？
3. 合作资源：结构生物学（cryo-EM）、AI/计算团队是否有合作方？
4. 时间规划：硕士还是博士？总共多少时间？
5. 经费预算：基因合成、高通量筛选的规模受限于经费
